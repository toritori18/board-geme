# Board Game Ranking

ボードゲームのランキングを作成・管理するWebアプリケーションです。
ユーザーがボードゲームを登録・評価し、スコアや人気度に基づいたランキングを表示します。

## 機能

- ボードゲームの登録・編集・削除
- ユーザーによる評価・レビュー
- スコア・人気度ランキング表示
