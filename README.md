# Board Game Ranking

ボードゲームを検索し、人気ランキングを閲覧できるWebアプリケーションです。

## 機能

- ボードゲームの検索（キーワード・プレイ人数・所要時間・難易度・ジャンルで絞り込み）
- 評価スコア順の人気ランキング表示
- ユーザー認証（ログイン・新規登録）
